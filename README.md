# Homebrew Tap for gitid

Custom Homebrew tap for [gitid](https://github.com/huvinesh/gitid) - Git identity switcher.

## Installation

```bash
brew tap huvinesh/gitid
brew install gitid
```

## Updating

```bash
brew upgrade gitid
```

## Uninstalling

```bash
brew uninstall gitid
brew untap huvinesh/gitid
```
